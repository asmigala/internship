1. Git and common commands - what is the purpose of following commands?
    - `git remote -v`
    - `git remote add upstream <repo-url>`
    - `git fetch`
    - `git pull`
    - `git rebase upstream/master master`
    - `git rebase -i HEAD~3`
    - `git push -f`
    - `git commit --amend`
    - `git reset --hard upstream/myBranch`
    - `git reset --soft origin/myBranch`
